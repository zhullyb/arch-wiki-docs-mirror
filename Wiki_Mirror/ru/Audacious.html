Audacious – свободный аудиоплеер, использующий GTK+. Он сосредоточен на качестве звука и поддерживает большое количество форматов аудио. Легко расширяется с помощью сторонних плагинов.

Установка

audacious можно установить с официального репозитория. Установка с помощью pacman: 

# pacman -S audacious

Настройка

Интерфейс

Примечание: GTK+ интерфейс используется по умолчанию, начиная с версии 2.4.

Audacious имеет два интерфейса:

  • интерфейс Winamp Classic;
  • интерфейс GTK.

Между ними можно свободно переключаться без перезапуска плеера.

Добавление скинов Winamp

Это делается очень легко. Просто скопируйте файл со скином (файл .zip, .wsz, .tgz, .tar.gz, или .tar.bz2) в каталог ~/.local/share/audacious/Skins (только для вашего пользователя) или в /usr/share/audacious/Skins (для всех пользователей), после чего найдите и выберите скин через меню Вид - Настройки интерфейса.

Воспроизведение Audio CD

Установите libcdio: 

# pacman -S libcdio

После этого Audacious сможет играть Audio CD.

Советы и рекомендации

Audtool

Audacious имеет мощный инструмент управления Audtool, с помощью которого можно получить информацию из плеера или управлять им.

Например, чтобы получить текущее название песни или исполнителя, введите следующую команду: 

$ audtool current-song
$ audtool current-song-tuple-data artist

Есть также функции для управления воспроизведением, эквалайзером, списком воспроизведения и главным окном программы. Весь список возможностей смотрите в помощи: 

$ audtool --help

Проблемы с MP3

Попробуйте установить mpg123: 

# pacman -S mpg123
