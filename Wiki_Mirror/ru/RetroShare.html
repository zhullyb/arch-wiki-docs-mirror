RetroShare — свободный клиент одноимённой Friend-to-Friend сети для приватного обмена файлами и сообщениями.

Установка из репозитория RetroShare

1. Добавьте репозиторий пакетов, отредактировав файл /etc/pacman.conf: 

[home_AsamK_RetroShare_Arch_Community_standard]
SigLevel = Never
Server = http://download.opensuse.org/repositories/home:/AsamK:/RetroShare/Arch_Community_standard/x86_64

2. Установите RetroShare. 

# pacman -Syu
# pacman -S home_AsamK_RetroShare_Arch_Community_standard/retroshare06

Желательно удалить все другие сборки RetroShare с вашего компьютера.

Установка из AUR

Текущая версия RetroShare доступна в AUR. Воспользуетесь пакетом retroshareAUR для сборки последней стабильной версии исходных текстов. Для этого, под root-пользователем выполните: 

# yaourt -Sa retroshare

Использование

После установки программа генерирует личный сертификат RetroShare, используя сведения, предоставленные пользователем. После этого происходит запуск программы. Так как соединения в F2F-сети устанавливаются только с доверенными участниками (пирами), то дальше пользователю необходимо обменяться сертификатом с действующим участником сети. Чтобы найти доверенных пиров, предлагается произвести обмен сертификатами с одним из публичных чат-серверов. После успешного обмена пользователю становятся доступны чат-комнаты сети RetroShare, в том числе чат "Key Exchange", предназначенный для обмена сертификатами с участниками сети.
