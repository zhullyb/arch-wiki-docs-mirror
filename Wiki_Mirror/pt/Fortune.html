fortune é um programa simples que exibe frases aleatórias comoventes, inspiradoras, tolas ou maliciosas de um banco de dados de citações. O utilitário de linha de comando fortune faz parte do pacote fortune-mod.

Instalação

Instale fortune-mod ou fortune-mod-gitAUR.

Dica:
  • Metapacotes que fornecem citações de fontes específicas estão disponíveis no AUR sob o prefixo "fortune-mod-". Por exemplo, fortune-mod-archlinuxAUR contém citações relacionadas ao Arch. Outro exemplo é fortune-mod-brAUR, que contém citações em português brasileiro.
  • Você pode usar wikiquote-fortuneAUR para gerar arquivos fortune de páginas do wikiquote, usando a sintaxe wikiquote-fortune X, sendo X o nome do pacote. Para usar esses arquivos, execute $ fortune /caminho/para/citação.

Uso

  • Linha de comando em um terminal:
$ fortune
It is Texas law that when two trains meet each other at a railroad crossing,
each shall come to a full stop, and neither shall proceed until the other has gone.
  • Para exibir uma citação aleatória ao iniciar um terminal interativo, adicione o comando fortune ao arquivo de configuração rc do seu shell preferido:
#!/bin/bash
# ~/.bashrc
fortune

#!/bin/dash
# ~/.dashrc
fortune
  • Para exibir uma citação aleatória ao fazer login em um terminal de login, adicione o comando fortune ao arquivo de configuração do perfil do seu shell preferido:
#!/bin/mksh
# ~/.profile
fortune
Nota: fortune exibe aspas e frases consideradas pelo seu mantenedor como não ofensivas. Aforismos podem ser ativados como uma mistura de ofensivo/não ofensivo ou apenas como saída potencialmente ofensiva. Veja fortuna(6)[link inativo 2020-07-31] para mais informações.

Criando arquivos personalizados

Para criar um arquivo personalizado, você deve primeiro criar um arquivo contendo as citações. Entre cada citação, você deve adicionar uma linha contendo apenas o caractere %. Salve esse arquivo usando o nome que desejar, por exemplo minhas_citacoes. Em seguida, execute strfile minhas_citacoes em um terminal, para permitir o acesso aleatório das aspas. Agora você pode acessar as citações executando fortune minhas_citacoes em um terminal. Para que este comando funcione, você deve estar no mesmo diretório que os arquivos minhas_citacoes e minhas_citacoes.dat. Se você deseja que suas cotações sejam acessíveis apenas executando o comando fortune, mova ambos arquivos para o diretório /usr/share/fortune/. Se você quiser fazer alterações no seu arquivo, repita desde o início.
