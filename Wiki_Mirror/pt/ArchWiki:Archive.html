Essa página existe como um alvo de redirecionamento para páginas arquivadas. Visite Category:Archive ou Special:WhatLinksHere/ArchWiki:Archive para listagens de tais páginas.

Se você chegou aqui após seguir um link para o que se esperava ser um artigo, isso significa que ele foi arquivado: você pode navegar suas revisões seguindo o link "Redirecionado de" logo abaixo do título desta página e, então, abrindo o histórico daquela página de redirecionamento.

Se entre as páginas arquivadas você encontrar algumas que faria sentido fazer um redirecionamento para um artigo existente, por favor faça isso.

Como arquivar uma página

Antes de arquivar uma página, considere redirecioná-la para um artigo existente mais adequado. se nenhum puder ser encontrado, siga esse procedimento:

  1. A página deve estar marcada com Template:Archive por pelo menos 1 semana.
  2. Remova todos os links (backlinks) que apontam para a página a ser arquivada ou um de seus redirecionamentos, procurando por sua página Special:WhatLinksHere.
  3. Se a página está nos espaços de nome "Template" ou "Category", ele deve ser desabilitado ao renomeá-lo, alterando o primeiro dois pontos (":") para um ponto e vírgula (";"). A página Talk associada, se existir, deve ser renomeada também. Por exemplo, "Template:Exemplo" seria alterada para "Template;Exemplo" e "Template talk:Exemplo" seria alterada para "Talk:Template;Exemplo".
  4. Substitua todo o conteúdo com um redirecionamento para esta página aqui, e categorize a página com Category:Archive: 
    #redirect [[ArchWiki:Archive]]
[[Category:Archive]]
    Tip: Isso pode ser feito mais rapidamente com Template:AR: 
    {{subst:AR}}
  5. Corrija qualquer redirecionamento duplo que possa resultar do arquivamento do artigo, isto é, todos os títulos que foram anteriormente redirecionados para ele devem ser igualmente arquivados como especificado na etapa 4.
  6. Se a página arquivada possui uma página Talk associada, esta deve ser arquivada com este mesmo procedimento, iniciando da etapa 2.
