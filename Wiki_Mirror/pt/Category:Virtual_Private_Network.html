Uma Rede Privada Virtual, do inglês Virtual Private Network (VPN), é um conceito de segurança para criar um canal de comunicação segura entre hosts por meio de outra conexão de rede (pública).

Essa categoria contém software de VPN geral, software de provedor específico pertencente a Category:VPN providers.

Veja também List of applications#VPN clients.
