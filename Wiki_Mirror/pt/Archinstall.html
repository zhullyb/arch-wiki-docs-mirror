archinstall é uma biblioteca auxiliar para instalar o Arch Linux. Ele é empacotado com diferentes instaladores pré-configurados, tais como o instalador "guided".

Este documento não fala sobre o uso do archinstall como uma biblioteca Python; para isso veja [1].

Atenção: archinstall é um programa experimental e oferece padrões diferentes do processo de Instalação comum. Quando usar um sistema instalado com archinstall, mencione-o nos pedidos de suporte e envie o /var/log/archinstall.log.
Nota: archinstall somente suporta sistemas UEFI.

Executando o instalador

Primeiro, baixe e inicialize o ambiente live como descrito em Guia de instalação#Pré-instalação. O pacote archinstall faz parte do ambiente live e pode ser executado diretamente: 

# archinstall

O instalador guided vai executar (perguntar) os seguintes passos:

  1. configurar o locale;
  2. selecionar os mirrors;
  3. particionar os discos;
  4. formatar as partições;
  5. habilitar criptografia de disco (opcional);
  6. definir o hostname;
  7. definir a senha do usuário root;
  8. instalar um gerenciador de boot (limitado ao systemd-boot).
Atenção: Deixar a senha do usuário root vazia desabilita a conta root, usando sudo para escalonamento de privilégios. Já que isto pode fazer você se trancar fora do seu sitema, isto nâo é geralmente sugerido. Veja Sudo#Desabilitar o login do root.
Nota: O instalador pode configurar redes a cabo no sistema instalado usando Systemd-networkd ou pode copiar a configuração usada na ISO. Isso quer dizer que, se você configurar uma rede sem fio usando Iwd#iwctl, sua configuração, incluindo a senha da rede, serão copiadas para o sistema instalado. Também copia a configuração para redes cabeadas que estão presentes na ISO.

Pacotes adicionais podem ser instalados ao especificá-los depois da mensagem Write additional packages to install ("Escreva os pacotes adicionais à instalar").

Uma vez que a instalação foi concluída, um texto verde deve aparecer dizendo que é seguro reiniciar, que é também o comando que você usa para reiniciar.

Perfis

archinstall inclui perfis (profiles), conjuntos de pacotes e opções pré-configuradas que podem ser instaladas após o sistema base.

Atenção: Perfis incluídos são específicos do archinstall e não são suportados pelos mantenedores de pacotes. Usuários são recomendados à verificar os detalhes de cada perfil antes de usá-lo.
