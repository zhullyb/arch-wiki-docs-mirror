WhatsApp é primeiro um serviço de mensagens instantâneas para dispositivos móveis pertencente ao Facebook, com criptografia de ponta a ponta. Requer um número de telefone celular ativo e conexão de dados para operar.

Não existe um cliente de desktop WhatsApp oficial para Linux, e o Facebook tem estritamente tentado banir clientes e plugins de terceiros usando seu protocolo. Devido a esse comportamento corporativo, recomendamos que as pessoas evitem usar o WhatsApp completamente em favor de serviços de mensagens instantâneas com mais abertura, como XMPP, Telegram ou ICQ.

Instalação

Você pode usar um dos métodos a seguir para usar o WhatsApp no Arch:

Clientes baseados na Web

  • WhatsApp Web, a única maneira oficial e legal de usar para usar o WhatsApp no Linux.
  • whatsapp-nativefierAUR, um wrapper de área de trabalho do WhatsApp Web construído usando o nativefier.
  • franzAUR, um aplicativo baseado em Chrome de código aberto que pode ser usado para interface baseada na web de vários softwares de mensagens instantâneas, incluindo o WhatsApp.
  • rambox-binAUR, como o Franz, um wrapper baseado na web para vários softwares de mensagens instantâneas, incluindo o WhatsApp.

Clientes Gráficos

  • voltAUR, um multiprotocolo relativamente novo que pretende ser nativo e suporta o WhatsApp.
