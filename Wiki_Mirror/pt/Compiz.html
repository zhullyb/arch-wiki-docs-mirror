Segundo a Wikipedia:

O Compiz é um gerenciador de janela de composição para o Xorg, que utiliza aceleração 3D para criar efeitos compostos rápidos de desktop para o gerenciador de janelas. Efeitos como minimizar e maximizar ou a área de trabalho em cubo são implementados como plugins carregáveis.

Instalação

A instalação pode ser feita usando os repositórios community. Uma outra forma seria utilizando os pacotes git no artigo Compiz Fusion Git.

Através do Repositório Community

Após habilitar o repositório no arquivo /etc/pacman.conf, pode-se instalar o compiz com o comando:

Todos os Pacotes

# pacman -S compiz-fusion

Pacotes GTK

# pacman -S compiz-fusion-gtk

Pacotes KDE

# pacman -S compiz-fusion-kde

Agora adicione os pacotes com os efeitos fusion: 

# pacman -S compiz-fusion-plugins-{main,extra}

Iniciando O Compiz Fusion

Com o Compiz Fusion Icon

$ fusion-icon

Agora clique com o botão direito no ícone da bandeja e 'selecione gerenciador de janelas'. Escolha "Compiz" (se já não estiver selecionado). Se falhar você pode iniciar manualmente com os seguintes comandos: 

$ fusion-icon
$ emerald --replace
$ compiz-manager

KDE

Manual Sem O Fusion-Icon

Inicie o Compiz com o seguinte comando: 

$ compiz --replace cpp&

Inicie o gerenciador de configurações e configure os efeitos desejados: 

$ ccsm&

Inicie o decorador de janelas: 

$ kde-window-decorator --replace

Automaticamente

Método 1 = Fusion-Icon

Crie um link do fusion-icon na pasta $HOME/.kde/Autostart: 

$  ln -s /usr/bin/fusion-icon ~/.kde/Autostart/fusion-icon
Método 2 = Manualmente criar um arquivo de inicialização automática 
$ kate $HOME/.kde/Autostart/compiz.desktop OU $ nano $HOME/.kde/Autostart/compiz.desktop
[Desktop Entry]
Encoding=UTF-8
Exec=compiz --replace ccp
StartupNotify=false
Terminal=false
Type=Application
X-KDE-autostart-after=kdesktop
