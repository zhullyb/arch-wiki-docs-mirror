Todo artigo deve ser categorizado: categorias são exibidas na parte inferior de cada artigo. Esse tutorial vai explicar como definir uma página de categoria, como criar páginas de categorias faltantes e como mover uma página para uma categoria diferente. Para as regras de estilo relacionado, veja Help:Estilo.

Artigos e categorias usadas como exemplos neste artigo não são reais. Se você não deseja praticar mover enquanto ler esse tutorial, encontre uma página real que você deseja mover.

Nota: Tudo que é discutido neste artigo se aplica às categorias em si quase que da mesma forma (i.e. categorias também podem ser categorizadas).

Definir categoria de página

Clique "Editar" no topo da página. Adicione o seguinte no início do texto: 

[[Category:Alguma categoria]]

Por favor, tente usar categorias existentes quando possível: você pode encontrar toda a árvore de categorias na Tabela de conteúdo (para categorias em inglês, veja Table of contents). Somente se não houver uma categoria adequada para seu novo artigo, salve a página com o nome da categoria inexistente, mas, em seguida, cria uma nova página de categoria seguindo #Criar nova página de categoria.

Criar nova página de categoria

Após ter salvado a página, você verá um link "Categoria: Alguma Categoria" na parte inferior do artigo. Se essa categoria não existir, ela será realçada de vermelho. Clique no link vermelho para ir para a categoria recém-criada.

Defina-a para ser uma subcategoria de uma categoria pai já existente: 

[[Category:Categoria pai]]

Finalmente, salve a página da categoria.

Se você acha que a nova categoria precisa de um pai que ainda não existe, por favor abra uma discussão na página talk da categoria em vez de criar você mesmo toda uma nova árvore de categorias.

Mover página para uma categoria diferente

Para mover uma página para uma categoria diferente, edite a linha [[Category:Alguma categoria]] da página.

Por exemplo, vamos mover uma página fictícia "ArchWiki Tutorial (Magyar)" para uma categoria fictícia apropriada. (Mover para categorias existentes é basicamente o mesmo que mover para categorias não existentes.)

Clique "Editar" no topo da página. Encontre a seguinte linha no início do texto: 

[[Category:Help]]

Altere para: 

[[Category:Help (Magyar)]]

Salve a página clicando no botão "Salvar alterações" logo abaixo da caixa de edição.

Nome de categoria i18n

Títulos de categorias seguem a mesmas regras descritas em Help:i18n (Português)#Títulos de páginas, mas note que elas não podem fazer uso de Help:i18n (Português)#Redirecionamentos localizados simplesmente porque MediaWiki não oferece suporte a eles.
