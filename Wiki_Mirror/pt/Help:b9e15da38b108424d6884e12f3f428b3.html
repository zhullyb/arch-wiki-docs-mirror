Esse artigo define os padrões para o uso de caracteres de espaço em branco no código fonte de artigos. O estilo usado nos exemplos é para serem considerados uma parte integral dos artigos.

Regras gerais

  • Use um único espaço para separar frases no mesmo parágrafo (i.e. após sinal de ponto).
  • Evite usar múltiplas linhas em branco para espaçar parágrafos ou seções.
  • Separe títulos de seção dos caracteres = com um caractere de espaço.
  • Separe cabeçalhos de seção de corpos de seção com uma linha vazia.
  • Não deve haver espaços em volta de nomes de template, parâmetros de template, títulos de link e texto alternativo de link.
  • Separe caracteres de lista (*, #, ; e :) de seus itens com um espaço. Quando usado para recuo, um espaço após : é opcional.
  • Itens de listas individuais não podem ser separadas por linhas em branco, do contrário o interpretador do wiki iniciará uma nova lista para cada item.
  • Separe blocos especiais (a caixa de artigos relacionados, listas, blocos de código, templates de status de artigos notes, tips, warnings etc.) de texto normal com uma linha vazia.
Dica: Veja as versões traduzidas desses blocos especiais para estes templates, além de outros templates, em Help:Modelo#Lista de templates.

Espaços de nome Talk, *_talk

  • Separe respostas diferentes com uma linha vazia.

Exemplo

== Discussion 1 ==

First post, paragraph 1.

First post, paragraph 2.

-- ~~~~

:Reply 1, single paragraph. -- ~~~~

::Reply 2, paragraph 1.
::Reply 2, paragraph 2.
::-- ~~~~

== Discussion 2 ==

First post, single paragraph. -- ~~~~

:Reply 1, paragraph 1.
:Reply 1, paragraph 2.
:-- ~~~~

::Reply 2, single paragraph. -- ~~~~
