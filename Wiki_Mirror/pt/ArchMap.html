Histórico

ArchMap foi originalmente criado por User:xterminus. Ele criou arquivos jpeg que poderiam ser baixados desta página. No entanto, ele não tinha mais tempo para mantê-los, então desistiu do projeto.

Quando o Google Earth foi lançado para Linux, User:brain0 recriou o projeto.

A terceira versão do ArchMap foi baseada no Google Maps.

A versão atual foi iniciada por User:Alux com ajuda de User:Kyrias e User:Fsckd.

archmap

archmap é um script python que pode gerar arquivos GeoJSON, KML e CSV analisados esta lista dos usuários do Arch Linux.

Nota: ArchMap está atualmente em desenvolvimento no GitHub. Se você tiver alguma sugestão, por favor envie-as nos fóruns ou crie um issue no repositório.

A documentação é hospedada por readthedocs.org.

Mapas

Arch Women hospeda arquivos GeoJSON, KML and CSV pré-formatados, bem como lista não tratada, reformatada que são feitos por archmap. Esses arquivos são atualizados quatro vezes ao dia às 00:00, 06:00, 12:00 e 18:00 UTC [1] e são usados para fazer os mapas abaixo.

OpenStreetMap

geojson.io é "uma ferramenta rápida e simples para criar, alterar e publicar mapas" que usa OpenStreetMap como uma camada base. Esse link vai baixar o arquivo GeoJSON do servidor do Arch Women e vai abri-lo para edição no site.

Google Earth

Você pode adicionar as coordenadas ao Google Earth permanentemente:

  • Clique com botão direito do mouse em Meus lugares
  • Vá em Adicionar -> Link de rede
  • Insira "ArchMap" no campo Nome, coloque https://archwomen.org/media/archmap/archmap.kml no campo Link e pressione OK.

Você pode atualizar os dados clicando com o botão direito do mouse na pasta ArchMap e selecionando Atualizar.

Mapas gerados pelo usuário

Este mapa foi criado por Eray Aydın em 24.08.2016

Este mapa é atualizado diariamente (cerca de 10:00am UTC) usando o script ArchMap. Os dados KML usados para renderizar o mapa estão disponíveis aqui[link inativo 2020-08-02 ⓘ], a lista não tratada de usuários está aqui[link inativo 2020-08-02 ⓘ].

Algumas outras renderizações dos dados são hospedadas por CartoDB e MapBox[link inativo 2020-12-19 ⓘ]. No entanto, eles podem estar desatualizados, pois precisam ser atualizados manualmente alux importando o arquivo GeoJSON.
