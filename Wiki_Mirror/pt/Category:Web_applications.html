Uma pilha tradicional de aplicativo web consiste em:

  • um sistema operacional (Linux)
  • um servidor web, como Servidor Apache HTTP ou nginx
  • um SGBD relacional, como MySQL ou PostgreSQL
  • uma linguagem de programação, como PHP, Python ou Perl

Um exemplo de tal pilha é o LAMP.
