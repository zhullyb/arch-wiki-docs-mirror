Navegadores web requisitam páginas web de servidores web usando o Hypertext Transfer Protocol (HTTP).

A versão segura do HTTP é HTTPS, que usa TLS.

A linguagem de marcação da web é a HyperText Markup Language (HTML), que é processada por navegadores da Web e pode ser escrita diretamente em um editor de texto, convertido de Markdown ou gerado com um editor WYSIWYG como o BlueGriffon.
