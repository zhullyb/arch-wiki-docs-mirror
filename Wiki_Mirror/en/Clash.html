Installation

Install the package clash.

Country.mmdb should be manually downloaded and placed at ~/.config/clash/Country.mmdb.

Subscription

Install clashupAUR and follow the instructions in GitHub.

GUI Management

Use docker image yacd to switch between profiles. Note that configuration is not preserved across the restart.
