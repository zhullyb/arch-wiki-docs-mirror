A traditional web application stack consists of:

  • an operating system (Linux)
  • a web server, like Apache HTTP Server or nginx
  • a relational DBMS, like MySQL or PostgreSQL
  • a programming language, like PHP, Python or Perl

An example of such a stack is XAMPP.
