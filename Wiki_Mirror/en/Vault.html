Installation

Vault can be installed with the package vault.

Server

Vault is controlled with the vault.service systemd unit. Enable and start this unit.

To initialize the vault after starting, see https://learn.hashicorp.com/tutorials/vault/getting-started-deploy#initializing-the-vault.

Configuration

Edit the /etc/vault.hcl file.

Running Server in dev mode

Follow https://www.vaultproject.io/docs/concepts/dev-server.html. Take note of API changes https://stackoverflow.com/questions/49872480/vault-error-while-writing.
