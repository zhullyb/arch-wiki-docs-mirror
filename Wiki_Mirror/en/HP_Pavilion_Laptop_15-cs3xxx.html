Installation

BIOS

First, disable secure boot in BIOS settings. If you want use netboot with UEFI ipxe (1.20.1) cannot obtain a lease.

You must use an USB key in UEFI or switch to legacy boot mode or use an USB ethernet adapter.

  • F9 boot menu
  • F10 BIOS settings

Inputs / Outputs

All function/multimedia keys working, backlight keyboard on 3 levels works.

Touchpad, camera, SD-Card reader, Ethernet, Wifi and Bluetooth works out of the box.

Known issues

  • ipxe in 1.20.1 version didn't support ethernet card.
