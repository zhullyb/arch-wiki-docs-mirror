This category is meant for general graphical user interface (GUI) articles, GUI applications belong in Category:Applications.

Desktop environment
Provides a complete GUI for a system by bundling together a variety of components.
Window manager
Is system software that controls the placement and appearance of windows within a windowing system in a GUI.
Display manager
Is a graphical login manager which starts a session on a display server.

For graphic card drivers, see Category:Graphics.
