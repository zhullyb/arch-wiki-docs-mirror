Vivaldi is a new web browser from former Opera founder & team members, based on Chromium and focused on personalization aspects.

Installation

Vivaldi can be installed with vivaldi or vivaldi-snapshotAUR. Prebuilt packages can alternatively be found in the herecura unofficial repository.

To use Qt instead of GTK dialogs for file selections just install kdialog.

Extensions

Vivaldi is compatible with most of Chrome's extensions. These can be installed directly from the Chrome Web Store. To see which extensions are installed/enabled, type vivaldi://extensions in the address bar.

See also Wikipedia:Google Chrome Extension.

Media playback

To enable proprietary media (H264, AAC, etc.) playback support:

  • install vivaldi-ffmpeg-codecs
  • start vivaldi fresh (after update or fresh install), close vivaldi and restart again
  • use the vivaldi-update-ffmpeg-hookAUR from AUR

If you still need flash you can install pepper-flash.
