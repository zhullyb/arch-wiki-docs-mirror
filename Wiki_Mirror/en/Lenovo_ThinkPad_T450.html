This article covers details pertaining to installing and configuring your Arch on Lenovo Thinkpad T450 laptop.

Installation

Supports both UEFI and MBR style bios. There are no issues with installing Arch Linux with the latest that is known Archiso. The Installation guide works except, the sound.

Check out the T450s wiki, almost everything from there works on the T450: Lenovo ThinkPad T450s

Note: Some T450 don't have a fingerprint scanner.

Sound

See ALSA#Set the default sound card to set the default sound card to Intel PCH (speakers and headphones). 

/etc/modprobe.d/thinkpad-t450s.conf
options snd_hda_intel index=1,0

Restart to set changes.
