Installation

Follow the installation guide, also see Dell 10v, Dell 1010, Dell 3531 or similar.

Quirks

Black screen at boot

Try blacklisting in /etc/modprobe.d/ during the chroot stage of the installation 

blacklist dell-laptop

Some key not working

Set the keyboard layout to "Dell Laptops Precision M"

No sound

Try 

amixer -D pulse set Mute Playback
