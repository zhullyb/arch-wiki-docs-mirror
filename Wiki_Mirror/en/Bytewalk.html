Installation

Install the bytewalkAUR package.

Smoke Test

Make sure that Bytewalk has been installed correctly by running the following in a terminal: 

$ bytewalk

This should print a help menu.

Troubleshooting

Reproducible bugs/errors may be reported on Gitlab.
