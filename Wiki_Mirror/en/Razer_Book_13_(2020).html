The Razer Book 13 (2020) is an ultrabook introduced in late 2020. It features a 13.4" screen (UHD or FullHD), 11th-gen Intel Core processors (i5 or i7) and integrated Intel Iris Xe graphics.

Firmware

Razer is not participating in fwupd. Which means that firmware updates must be done using Windows.

Speakers

To make speakers work you have to install sof-firmware.

Keyboard

Keyboard lighting effects can be controlled using openrazerAUR. For more information visit Razer peripherals.
