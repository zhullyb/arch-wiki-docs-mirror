Otter Browser is a web browser which aims to recreate Opera UI. It's based on Qt5 and QtWebKit, with experimental support of QtWebEngine.

Installation

Otter Browser can be installed with one of the following packages:

  • otter-browser - the stable release built with the QtWebKit backend
  • otter-browser-webengineAUR - the stable release built with the experimantal QtWebEngine backend
  • otter-browser-gitAUR - latest development snapshot

Tips and tricks

Experimental support for QtWebEngine

To use QtWebEngine based on Blink web engine, go to about:config and select qtwebengine as Backend.
