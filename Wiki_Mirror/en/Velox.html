Velox is a dynamic tiling window manager for Wayland. It is currently under development.

Installation

Install the veloxAUR package, or velox-gitAUR for the development version.

The default terminal emulator for Velox is st. Other terminals such as urxvt also work, but also require xwayland.

You probably also want to install dmenu to be able to more easily launch applications.

Configuration

Velox is configured from a configuration file. The configuration file should be in the users home directory named .velox.conf and the default configuration can be found here.

Starting Velox

If you are using the veloxAUR package, you simply execute velox from your terminal.
