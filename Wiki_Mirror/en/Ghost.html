The printable version is no longer supported and may have rendering errors. Please update your browser bookmarks and please use the default browser print function instead.

Ghost is a free and open source blogging platform written in JavaScript and distributed under the MIT License, designed to simplify the process of online publishing for individual bloggers as well as online publications.

Installation

Install ghost-webAUR, and modify /etc/webapps/ghost/config.js. Then start ghost.service. If you are happy with it, enable it for automatic start when the system boots.

Visit http://127.0.0.1:2368/ghost for final configuration.
