Brook is a cross-platform strong encryption and not detectable proxy. It's goal is to keep it simple, stupid and not detectable.

Installation

Install the brook package.

Usage

To start brook as ws server: 

# brook wsserver -lÂ :<port> -p <password>

To start brook as ws client: 

# brook wsclient -s ws://<server_ip>:<port> -p <password> --socks5 <client_ip>:<port>
