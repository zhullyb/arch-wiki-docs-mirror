Konsole is a terminal emulator for the KDE desktop.

Installation

Install the konsole package.

Troubleshooting

True-color programs do not display correctly

This is because, by default, konsole does not set the right $TERM variable. To set it, either edit you konsole profile or make new one: 

~/.local/share/konsole/konsole.profile
[General]
Name=konsole
Environment=TERM=konsole-direct,COLORTERM=truecolor
