Running swift-language builds and doing swift development is possible on linux. To install swift use swift-language-gitAUR for latest from git, or swift-binAUR for the released version. For code completion the sourcekit language server protocol, sourcekit-lsp is available, e.g. as plugin to visual studio code, vscode-oss[broken link: package not found].

hello world

swift package manager allows to create example programs. 

hello-world executable
swift package init --type executable
swift run

hello-world library
swift package init
swift build
