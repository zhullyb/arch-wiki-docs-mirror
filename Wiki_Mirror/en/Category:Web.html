Web browsers request web pages from web servers using the Hypertext Transfer Protocol (HTTP).

The secure version of HTTP is HTTPS, which uses TLS.

The markup language of the web is the Hypertext Markup Language (HTML), which is rendered by web browsers and can be written directly in a regular text editor, converted from Markdown or generated with a WYSIWYG editor like BlueGriffon.
