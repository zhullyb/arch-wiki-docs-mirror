Questa categoria include articoli relativi alle applicazioni peer-to-peer, client bit-torrent, client di chat ed altre applicazioni relative alle connessioni internet.

Programmi di interconnessione fra reti, autenticazione, o servizzi specializzati solitamente si trovano nella categoria Networking. La categoria Web server contiene articoli di vari tipi di server: server web, mail server, server per repository, et cetera.
