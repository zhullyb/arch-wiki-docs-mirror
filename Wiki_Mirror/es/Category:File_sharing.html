Maneras de compartir archivos:

  • Memoria USB / unidad externa con lsblk y mount
  • a través de la red
    • cifrado siempre: SFTP, FTPS, HTTPS
    • cifrado opcionalmente: BitTorrent, SMB, WebDAV
    • sin cifrado: FTP, HTTP, NFS, netcat
  • con un sistema de control de versiones como Git o Mercurial
  • con un programa de sincronización como rsync, Syncthing o Unison
  • con la nube (servicio de internet propio o externo)
  • con Bluetooth
