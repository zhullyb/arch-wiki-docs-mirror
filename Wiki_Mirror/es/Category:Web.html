Los navegadores web solicitan páginas web desde servidores web utilizando un protocolo de transferencia de hipertexto (HTTP).

La versión segura de HTTP es HTTPS, que utiliza TLS.

El lenguaje de marcas de la web es Lenguaje de Marcas de Hipertexto (HTML), el cual es renderizado por los navegadores web y puede ser escrito directamente en un editor de textos normal, convertido desde Markdown o generado con un editor visual como BlueGriffon.
