El comando version es una creación reciente con dos objetivos en mente

  1. Leer versión de un paquete
  2. Operar con la versión de un paquete

El programa puede ser instaldo desde el repositorrio AUR, una detallada guia de instalcion la puede encontar en la seccion de instalcion mas abajo.

Instalacion desde AUR

Si no conoce la mecánica de instalar paquetes desde AUR es recomendable leer primero Arch User Repository (Español)

El paquete a instalar es octetos-version, sin embargo, depende de octetos-coreutils, que a su vez depende de octetos-core. Por lo que deverá instalar en el siguiente orden: 

 Instalar desde AUR octetos-core
 Instalar desde AUR octetos-coreutils
 Instalar desde AUR octetos-version

Algunos de los paquetes tienen dependencias en los repositorios de pricipales por lo que deverá de aceptar los paquetes para continuar el proceso.

Uso del comando

Puede saber la versión de pacman con: 

 $ version -g pacman

Optendrá una salida limpia y especifica como la siguiente: 

 5.2.2

Pueve validar si la versión cumple con un críterio, por ejemplo: 

 $ version -c pacman g= 2.0

o si su consola acepta una sintaxis mas amigable 

 $ version -c pacman >= 2.0

ambos comandos retornaran 

 true

una salida mas significativa se puede optener mediante la opcion 'i' 

 $ version -ci pacman g= 2.0
 pacman : pass(5.2.2)

también puede obtener información del plataforma actual con lo opción 'p' 

 $ version -p
 Pacman

Puede encontrar un ejemplo mas complejo en el subdirectorio de examples en el código fuente.
