Nitrogen es un navegador y configurador de fondos de escritorio rápido y ligero para Xorg.

Instalación

Instale el paquete nitrogen.

Utilización

Ejecute la orden nitrogen --help para obtener todos los detalles. Los siguientes ejemplos le ayudarán a empezar:

Establecer un fondo de escritorio

Para ver y establecer el fondo de escritorio deseado desde un directorio específico de forma recursiva, ejecute: 

$ nitrogen /path/to/image/directory/

Para ver y establecer el fondo de escritorio deseado desde un directorio específico de forma NO recursiva, ejecute: 

$ nitrogen --no-recurse /path/to/image/directory/

Restaurar el fondo de escritorio

Para restaurar el fondo de escritorio elegido durante las siguientes sesiones, inicie automáticamente la siguiente orden: nitrogen --restore &

Solución de problemas

Congelación con monitores duales

Elimine la configuración actual de Nitrogen: [1] 

$ rm -r ~/.config/nitrogen/
