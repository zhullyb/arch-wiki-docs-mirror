Introducción

File Alteration Monitor (FAM) es un demonio usado por entornos de escritorio como GNOME y Xfce para monitoreo y reporte de cambios en el sistema de archivos. FAM también es usado por el servidor Samba. KDE ya no utiliza FAM, han migrado al sistema inotify del kernel, el cual no requiere ninguna configuración.

Gamin es una reimplementación de las FAM. Es mas nuevo y su mantenimiento es mas activo que el de FAM, también es mas facíl de configurar.

Instalación

Normalmente FAM es instalado como una dependencia de otro paquete. Puede ser directamente instalado con: 

pacman -S fam

Configuración

Se puede inicial FAM manualmente con: 

/etc/rc.d/fam start

Para iniciar FAM al arrancar, agrega fam en la línea DAEMONS en rc.conf. Sí utilizas NetworkManager fam debe estar después de networkmanager
