Esta categoría está destinada a los artículos generales de interfaz gráfica de usuario (GUI), las aplicaciones GUI pertenecen a la categoría de aplicaciones.

Entorno de escritorio
Proporciona una GUI completa para un sistema agrupando una variedad de componentes.
Administrador de ventanas
Es un programa del sistema que controla la ubicación y el aspecto de las ventanas dentro de un sistema de ventanas en una GUI.
Administrador de pantalla
Es un administrador de inicio de sesión que inicia una sesión en un servidor de visualización.

Para los controladores de tarjetas gráficas, consulte la categoría sobre gráficos.
