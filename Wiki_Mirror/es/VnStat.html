vnStat es un monitor de tráfico de red ligero (de línea de comandos). Supervisa interfaces seleccionables y almacena los registros de tráfico de red en una base de datos para su posterior análisis.

Instalación

Instale el paquete vnstat.

Configuración

Inicie/Active el demonio vnstat.service.

Escoja una interfaz de red preferida y edite la variable Interfaz en /etc/vnstat.conf. Para listar todas las interfaces disponibles para vnstat, utilice vnstat --iflist.

Para empezar a monitorizar una interfaz concreta que no estaba referida en el archivo de configuración cuando se inició el demonio, primero debe inicializarse una base de datos. Cada interfaz necesita su propia base de datos. La orden para inicializar una para la interfaz eth0 es: 

 # vnstat --add -i eth0

Recuerde reiniciar el demonio vnstat.service después de añadir una nueva interfaz.

Utilización

Para consultar el tráfico de la red, ejecute la orden: 

 # vnstat --query

Puede observar el tráfico de la red en vivo mediante: 

 # vnstat --live

Para encontrar más opciones, use: 

 # vnstat --help

O para ver todas las opciones, use: 

 # vnstat --longhelp

También es posible conseguir presentar de manera atractiva los datos con vnstati(1), el cual forma parte del paquete vnstat.
