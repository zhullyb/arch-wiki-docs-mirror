翻译状态：本文是 SonyEricsson samba sharing 的翻译。上次翻译日期：2020-08-18。如果英文版本有所更改，则您可以帮助同步翻译。

您可以通过 samba 共享访问 Sony Ericsson 手机和电话卡。它即使在使用旧固件的情况下也可以使用，并且您仍然拥有功能齐全的手机，而不像 USB 大容量存储模式。

支持的手机

  • Se C702

设置手机

  • Settings > Connectivity > USB > USB network > USB Network type > Via Computer
  • Settings > Connectivity > Network sharing > set up name and password
  • Settings > Connectivity > Sreaming settings > Connect using > USB Ethernet（设置 IP 地址，例如 192.168.0.50，掩码 255.255.255.0）
  • Settings > Connectivity > Sreaming settings > Connect using > USB Ethernet > Allow local connection > Yes

设置计算机

  • 您必须安装 Samba。
  • 启动 samba:
# /etc/rc.d/samba start
  • 在 Phone Mode 下插入手机
  • 设置手机的 IP：
# ip link set dev usb0 address 192.168.0.50
  • 现在您应该可以 ping 您的手机
$ ping 192.168.0.50
  • 例如在 Konqueror，您可以输入以下内容来浏览手机
smb://192.168.0.50

输入您在手机中选择的名称和密码
