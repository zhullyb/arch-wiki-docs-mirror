GNU Compiler Collection（GCC）是 GNU 工具链的一部分，并包括 C 和 C++ 的前端。

安装

安装 gcc 包。

其他可用的前端有：

  • gcc-ada Ada 前端
  • gcc-d D 前端
  • gcc-fortran Fortran 前端
  • gcc-go Go 前端
  • gcc-objc Objective-C 前端
