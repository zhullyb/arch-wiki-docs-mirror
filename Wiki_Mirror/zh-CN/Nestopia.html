Nestopia 是一个开源的 NES 模拟器，它试图尽可能准确地模拟 NES 硬件。它适用于 Windows，macOS，Linux 和 BSD。也有一个 libretro 端口，有关更多信息，请参见 RetroArch。

安装

从 AUR 安装 nestopiaAUR 包.。或者，安装 nestopia-gitAUR 开发版本。

故障排除

libao 后端无音频

目前，libao后端已损坏。建议在 Emulator -> Configuration -> Audio 中将音频 API 更改为 SDL。
