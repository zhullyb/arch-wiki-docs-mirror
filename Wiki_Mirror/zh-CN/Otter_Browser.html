Otter Browser 是旨在重新创建 Opera UI 的 Web 浏览器。它基于 Qt5 和 QtWebKit，并具有 QtWebEngine 的实验支持。

安装

Otter Browser 可以使用以下软件包之一安装:

  • otter-browser - 具有 QtWebKit 和 QtWebEngine 支持的稳定版本
  • otter-browser-nowebengine[断开的链接：replaced by otter-browser] - 没有 QtWebEngine 支持的稳定版本
  • otter-browser-gitAUR - 最新开发快照

提示和技巧

对 QtWebEngine 的实验支持

要使用基于 Blink web engine 的 QtWebEngine，请转到 about:config 并选择 qtwebengine 作为 Backend.
