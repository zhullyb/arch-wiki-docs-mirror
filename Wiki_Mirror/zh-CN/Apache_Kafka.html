Apache Kafka 是一个分布式流平台，可以：

  1. 使您可以发布和订阅记录流。在这方面，它类似于消​​息队列或企业消息传递系统。
  2. 使您可以以容错的方式存储记录流。
  3. 使您可以在记录流出现时对其进行处理。

安装

安装 kafkaAUR 包。

使用 systemctl 启动 kafka.service，它也应该自动启用/启动 zookeeper@kafka.service。

用法

有关用法，请参阅官方文档

客户端

  • C - librdkafka-gitAUR
  • Python - https://github.com/dpkp/kafka-python
  • Php - php-rdkafkaAUR
  • Perl - perl6-pkafkaAUR
