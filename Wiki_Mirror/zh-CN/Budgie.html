Budgie 是 Solus OS 的默认桌面环境。除了采用更现代的设计，Budgie 还可以模仿 GNOME 2 桌面的外观和质感。

安装

安装 budgie-desktop 软件包获取最新稳定版 Budgie 桌面，或者安装 budgie-desktop-gitAUR 获取当前 git 主分支版本。建议安装其所有可选依赖以获得更完整的桌面体验。同时也建议安装 gnome 软件组，其中包含标准 GNOME 体验所需的应用程序。

启动

从 显示管理器 桌面菜单中选择 Budgie Desktop 会话，或者修改 xinitrc 使其包括 Budgie 桌面： 

~/.xinitrc
export XDG_CURRENT_DESKTOP=Budgie:GNOME
exec budgie-desktop

使用方法

Budgie 的侧栏被称作 "Raven"，可以在上面查看通知消息，设置音量，以及修改桌面的外观等。可以通过按下 Super+N 键或点击 状态指示器 托盘小程序来访问它。

配置

Budgie 桌面的配置可以通过侧栏来完成，改变系统设置则通过 gnome-control-center 来完成。

改变按钮布局

使用 dconf、dconf-editor 或者 gsettings 可改变窗口按钮布局。

比如： 

gsettings set com.solus-project.budgie-wm button-layout 'close,minimize,maximize:appmenu'
gsettings set com.solus-project.budgie-helper.workarounds fix-button-layout 'close,minimize,maximize:menu'

启用锁屏

要让系统在不活动一段时间后启用锁屏，请安装 gnome-screensaverAUR。

使用不同的窗口管理器

The factual accuracy of this article or section is disputed.

Reason: Many window managers do not support --replace. (Discuss in Talk:Budgie (简体中文)#)

Budgie 可以使用其它的 窗口管理器。定义一个 自定义 GNOME 会话，用另一个窗口管理器替换 budgie-wm，或者将 my-wm --replace 命令设为自动启动，其中 my-wm 是要使用的窗口管理器的可执行文件名。
