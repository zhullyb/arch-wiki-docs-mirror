Clojure 是 Lisp 的方言，它是一种针对 Java 虚拟机，CLR 和 JavaScript 的动态，功能齐全的通用编程语言。

安装

安装 clojure 包，或安装 clojure-gitAUR 开发版本。

REPL

要运行 REPL，请从 leiningen 安装 leiningen。然后在终端运行 

lein repl

m2 仓库路径

要配置本地 m2 仓库路径，请修改profiles.clj: 

{:user {:local-repo #=(eval (str (System/getenv "XDG_CACHE_HOME") "/m2"))
        :repositories  {"local" {:url #=(eval (str "file://" (System/getenv "XDG_DATA_HOME") "/m2"))
                                 :releases {:checksum :ignore}}}
        }}
