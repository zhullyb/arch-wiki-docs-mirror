型号版本 日期 显示 声音 以太网 无线网络 蓝牙 电源管理 调制解调器 其他 备注
MacBook Pro Rev3 x86_64 2007.08 是 是 是 是 是 Cpufreq + pm-utils 未使用 – –
MacBook Mid2007 i686 2008.03 是 是 是 是 未使用 Cpufreq + laptop-mode-tools 未使用 – 挂起至 RAM 正常
MacBook Mid2007 x86_64 2009.08 是 是 是 是 是 pm-utils 未使用 – 开箱即用，iSight 除外
MacBook Pro 5,2 17" x86_64 2009.08 是 是 是 是 未使用 cpufreq + pm-util 未使用 – 暂停，进入休眠状态
Macbook Air 2013 x86_64 2014.10.01 是 是 是 是 是 开箱即用 未知 – 除网络摄像头外，其他所有功能均可使用（不支持网络摄像头）
Macbook Mid2006 i686 2014.10.01 是 是 是 是 使用 HID 到 HCI 转换 开箱即用 – synaptics 驱动程序允许三次单击，这是 OSX 中不提供的功能。 除网络摄像头外的所有内容，请使用 isight-firmware-tools
Macbook Pro mid-2009 x86_64 2016.08.01 是 是 是 是 是 未知 – 从具有专有 nvidia 驱动程序的图形会话切换时，虚拟控制台不会显示。在 nouveau 驱动程序上不会发生这种情况。 除 iSight 外的所有东西都可以直接使用
Macbook Pro mid-2010 x86_64 2017.04.01 是 是 是 是 是 未知 – 一切似乎都正常。但是，存在一些未解决的热量问题。这些可以用 mbpfan-gitAUR 解决 broadcom-wl 驱动用于 wifi
MacBook Pro (Mid-2012) x86_64 2018.03.01 是 是 是 是 是 laptop-mode-tools – – 您将需要 Eth 或 Teather 来安装 BCM 无线驱动程序，因为它们是非自由的且不随内核一起提供，有关更多信息，请参阅 MacBookPro9,2 (Mid-2012)#Wireless。
