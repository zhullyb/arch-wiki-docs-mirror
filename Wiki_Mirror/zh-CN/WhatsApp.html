WhatsApp 是 Facebook 拥有的移动优先即时消息服务，具有端到端加密功能。它需要有效的手机号码和数据连接才能运行。

没有适用于 Linux 的官方 WhatsApp 桌面客户端，Facebook 严格尝试禁止使用其协议的第三方客户端和插件。由于这种公司行为，我们建议人们避免完全使用 WhatsApp 来支持具有更开放性的 IM 服务，例如 XMPP，Telegram 或 ICQ。

安装

您可以使用以下方法之一来在 Arch 中使用 WhatsApp：

基于 Web 的客户端

  • WhatsApp Web，这是在 Linux 上使用 WhatsApp 的唯一官方方式。
  • whatsapp-nativefierAUR，是使用 nativefier 构建的 WhatsApp Web 的桌面包装。
  • franzAUR，这是一个基于 Chrome 的开源应用程序，可用于各种即时消息传递软件（包括 WhatsApp）的基于 Web 的界面。
  • rambox-binAUR，类似于 Franz，它是基于 Web 的包装程序，用于各种即时消息传递软件（包括 WhatsApp）。
